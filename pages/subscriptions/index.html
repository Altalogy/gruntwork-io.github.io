Subscriptions

A better way to manage your infrastructure.

  • Let Gruntwork handle updates to the generic components in your infrastructure.
  • Lean on Gruntwork engineers when you run into a tricky DevOps or AWS problem.
  • Use our automatic updates to keep your infrastructure code in line with the latest best practices.

Enjoy these benefits by becoming a Gruntwork member.

Get full access to the Gruntwork Library.

Members receive full access to the Gruntwork Library, a collection of standalone modules maintained and supperted by Gruntwork. A module is a reusable, battle-tested solution to a common infrastructure problem, such as how to run a Docker Cluster or how to configure your Network Topology, written using a mixture of Terraform, Go, Bash, and Python.

Browse the Gruntwork Library

Get Gruntwork Library updates.

As a Gruntwork Member, you get access not only to all the Gruntwork Library modules we have today, but also additions, updates, and fixes we make the to the Gruntwork Library in the future, for the lifetime of your membership.

AWS, Terraform, Docker, and all other DevOps technologies are changing all the time. Keeeping up with all of this is the core of our business, and as we keep working new clients and keep improving our practices, we capture all of that work as new modules and versioned updates to existing modules. As a member, you can leverage all of this work with a simple version of number bump.

Get automated infrastructure updates.(Beta)

One suprising truth we learned early on is that almost no one keeps their infrastructure up to date. Worse, as your infrastructure grows, it becomes exponentially more difficult. That’s why we created Gruntwork Patcher, a service that automatically scans your current infrastructure code, computes the delta between your current code and your desired update, and auto-generates the code updates, and auto-submits a Pull Request.

Automated infrastructure updates are currently in beta and available upon request with certain limitations.

Get support.

As a subscriber, you get access to a team of DevOps experts for a tiny fraction of the cost of hiring a single DevOps engineer full time. We are here to help you with:

  • Questions
  • Troubleshooting
  • Code reviews
  • Design reviews
  • Bug fixes
  • Updating your existing infrastructure
  • Building a new infrastructure
  • Office hours

Get access to the Gruntwork Community Forum.

Members get access to the Gruntwork Community Forum where you can talk with the other Gruntwork customers. Since you are all using the same infrastructure, this is a great place to ask questions, discuss design decisions, and work together as a community to improve infrastructure and DevOps practices.

Receive the Monthly Newsletter.

We invite you to mentally check out from Terraform, AWS, and DevOps industry updates for one month at a time, until we send our Monthly Newsletter! It’s a description of all the updates we’ve made the last month, how to update your infrastructure to use them, as well as general news on what’s happening in the AWS, Terraform, and Gruntwork ecosystem.

Receive security alerts.

New security vulnerables are discovered all the time. Rarely does a month go by without at least one critical vulnerability being announced in OpenSSL, PHP, Jenkins, WordPress, or the Linux kernel. We monitor a number of security advisory lists and notify you immediately if any critical vulnerabilities are announced that may affect your infrastructure.
