How it Works

  Watch the Overview Video

1. Tell us your infrastructure needs.

Using a Docker cluster? Which one? Since AWS account or multiple AWS accounts? Which environments do you need? Use our simple online form and choose from a standard set of common options. Don’t see what you need? Let us know and we can build it for you.

Learn More

2. We write your code.

We’ll write the code for an entire end-to-end reference architecture that implements the infrastructure you selected. The code incorporates a wealth of best practices:

  • Written in HCL (Terraform), Go Bash, and other industry-standard open source languages
  • Backed by automated tests
  • Well-documented
  • Built on Top of the Gruntwork Library to maximize code reuse
  • Fully customized to your team and unique needs

Learn about Reference Architecture
View Gruntwork Library

3. Schedule a live walkthrough.

We’ll show you the actual code you’ll use, applied live to a sample AWS account. If it looks good, complete your payment, and we’ll transfer the code to your git repos. If you don’t like it, pay nothing. You’ll have access to 100% of the code used to create your stack.

Watch our detailed walkthrough that covers all aspects of your new infrastructure, or review a comprehensive written guide that lives in your git repo. Need help with the foundations? Sign up for additional training on Terraform, Packer, Docker, security best practices, and more.
