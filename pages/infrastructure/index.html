What’s an infrastructure package?

At Gruntwork we’ve taken the thousands of hours we spent building infrastructure on Amazon Web Services (AWS) and condensed all that experience and code into pre-assembled Infrastructure Packages. An Infrastructure Package is a reusable, battle-tested, documented, configurable, best-practices definition of a piece of Infrastructure.
