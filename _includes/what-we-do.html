What we do

At Gruntwork, we've taken the thousands of hours we spent building infrastructure on AWS for our clients and condensed all that experience and code into a library of infrastructure-as-code packages that are thoroughly tested, documented, proven in production, and reusable. We work with your company to adapt the packages to your needs and get you set up with world-class infrastructure and DevOps practices in about 2 weeks.

Some of our most popular packages are:

Continuous integration

Automated build pipeline that runs on a CI server (e.g. Jenkins, Travis CI, Circle CI) after every commit to build your code, run your tests, and produce a versioned artifact of your app (e.g. Docker image or AMI built via Packer).

Automated deployment

Zero-downtime, immutable deployment triggered via a single click in your CI server, a commit to a specific branch, or even after every commit if you prefer continuous deployment. Includes integration with health checks and automatic rollback.

Network security

A secure VPC configuration with multiple environments (staging, production, mgmt), multiple layers of subnets (public, private, persistence), strict security groups, NAT gateways, and a bastion host for secure SSH access.

Secrets management

Set up Vault for secrets management. Use Vault for configuration secrets (e.g. DB passwords), generating temporary SSH keys (instead of storing unencrypted Key Pairs), and generating temporary AWS credentials (instead of storing unencrypted access keys).

Monitoring

Centralized log aggregation, metrics for all of your servers (e.g. CPU usage, memory usage, throughput, latency), and automatic alerts for a variety of problems.

Availability & Scalability

High availability and scalability using Auto Scaling Groups, Auto Scaling Policies, Elastic Load Balancers, S3, CloudFront, and automated data backup & restore.

Configuration

Configure your infrastructure using Terraform; configure your apps at build time (e.g. dependencies) using Docker and Packer; configure your apps at runtime (e.g. service discovery) using Consul.

Consulting

Don't see a package that does what you need? We offer general DevOps consulting on an hourly basis.
